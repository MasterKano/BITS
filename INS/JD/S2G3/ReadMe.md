## 12-session plan (overview)

1. Session 01 — stance, central problem, pummelling core  
2. Session 02 — hip jousting, over-under fundamentals  
3. Session 03 — entries (legs → upper body), shot defense → upper body grips  
4. Session 04 — front headlock: snapdowns, control, basic finishes  
5. Session 05 — cow catcher series + front headlock takedowns  
6. Session 06 — sacrifice throws from head/arm: sumi gaeshi, ude gaeshi  
7. Session 07 — trip package from head/arm: ouchi, kosoto, kouchi  
8. Session 08 — double unders + body lock takedown package  
9. Session 09 — pinch headlock + high-amplitude throw options  
10. Session 10 — older athlete throwing package (low-impact, high-control)  
11. Session 11 — build individual A-games from the system (Person A / Person B)  
12. Session 12 — system integration: mixed entries, throws, live games
